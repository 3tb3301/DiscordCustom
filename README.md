## التثبيت

> [!WARNING]
> This next command may need to be run as admin/root depending on your system, and you may need to close and reopen your terminal for pnpm to be in your PATH.
```bash
npm i -g pnpm
```

> [!IMPORTANT]
> Make sure you aren't using an admin/root terminal from here onwards. It will mess up your Discord/Equicord instance and you will most likely have to reinstall.

### Clone Equicord:
```bash
git clone https://github.com/3tb3301/DiscordCustom
cd Equicord
```

### Install dependencies:
```bash
pnpm install --frozen-lockfile
```

### Build Equicord:
```bash
pnpm build
```

### Inject Equicord into your desktop client:
```bash
pnpm inject
```
